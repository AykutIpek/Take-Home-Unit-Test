
# Take Home App


It is a restful api project compatible with full SOLID principles, where we can add new people and examine their details.

## Used technologies

**Client:** SwiftUI

**Tests:** Unit Test, UI Test

**API Type:** Restful

**API Source:** 

```http
https://reqres.in
```

```http
/api/users
```
